# Automated Server Health Check

## Project Overview
This project automates server health checks and sends notifications if any issues are detected.

## Technologies
- Bash
- curl
- mail

## Setup
1. Modify the script with your server's IP address.
2. Set up email configuration for notifications.
3. Run `chmod +x health_check.sh` to make the script executable.
