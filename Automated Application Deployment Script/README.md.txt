# Automated Application Deployment Script

## Project Overview
This project automates the deployment of a web application using Git and Docker.

## Technologies
- Bash
- Git
- Docker

## Setup
1. Modify the script with your repository and Docker settings.
2. Run `chmod +x deploy_app.sh` to make the script executable.
